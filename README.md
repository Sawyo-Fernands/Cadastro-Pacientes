# Cadastro-Pacientes


## Contexto

A clínica ACME deseja ter uma solução web para consultar e cadastrar seus pacientes.

## Funcionalidades

- Filtro por nome
- Listar pacientes
- Cadastrar pacientes
- Inativar pacientes

## Tecnologias

- Typescript
- ReactJS

## Execução

- Executar o comando yarn ou npm
- Executar o comando  yarn dev ou npm dev
